# 概要

- AIプロンプトを用途別に、ディレクトリで分けたテンプレート集

# ファイル・ディレクトリ一覧

| パス               | 説明                                          |
| ------------------ | --------------------------------------------- |
| `readme/`          | README作成に関するファイルを格納するディレクトリ |
| `readme/prompt.md` | README作成のためのルールを定義したプロンプト       |

# 使用方法

- リポジトリをクローンする
- 対象の用途に応じたディレクトリに移動する
- プロンプトファイルを参照し、必要に応じて編集して利用する
  - `ln -s xxx/prompt.md ./prompt.md`
    - このリポジトリのテンプレートファイルを、シンボリックリンクとして、配置し、AIエージェントの入力に使用する。
